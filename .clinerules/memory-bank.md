## Brief overview
Rules specific to the FoodLister project regarding memory bank usage. These guidelines ensure project context is properly loaded before starting any new task.

## Memory bank requirement
- When starting a new task, you MUST read ALL files in the `memory-bank/` directory before proceeding with any work
- The memory bank files are located in the `memory-bank/` folder at the project root
- Required files to read:
  - `memory-bank/activeContext.md`
  - `memory-bank/productContext.md`
  - `memory-bank/progress.md`
  - `memory-bank/projectbrief.md`
  - `memory-bank/systemPatterns.md`
  - `memory-bank/techContext.md`
- This step is mandatory and must be done at the beginning of every new task to ensure full context awareness
